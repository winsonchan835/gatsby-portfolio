A fully deployed iOS Application that showcases a fitness coach's multimedia posts. This application was developed with React Native and AWS S3, Cognito and DynamoDB. A user can navigate to multiple categories, view any media they would like to, and save any post into their favorites. Apple In-App Purchases are fully integrated into the app for new Users to seamlessly create new accounts. Currently only on iPhone.
